Bantu Saya membuat semua kelas yang Saya minta untuk membangun sistem manajemen jendela kustom di JavaFX. Berikut adalah ringkasan dari semua komponen yang akan dibuat:
1. Package id.alphareso.meidofx.base.controls

TitleBarControls: Menyediakan tombol kendali jendela (minimize, maximize, close) dengan tampilan yang dapat disesuaikan dan sudah memiliki fungsi kendali jendela.
referensi ikon kendali jendela:
images/close-light.png
images/minimize-light.png
images/maximize-light.png

2. Package id.alphareso.meidofx.base.enums

TitleBarStyle: Enum yang mendefinisikan berbagai konfigurasi tampilan untuk title bar dengan pilihan:

ALL: Menampilkan semua komponen (ikon, judul, dan tombol kendali)
NO_LEFT: Hanya menampilkan tombol kendali
NO_RIGHT: Hanya menampilkan ikon dan judul
NO_ALL: Tidak menampilkan apa-apa

untuk menggunakannya saya ingin seperti ini: 
public static RoundStage stage;
TitleBar titleBar=new TitleBar(Main.stage, TitleBarStyle.ALL);


3. Package id.alphareso.meidofx.base.handlers

TitleBarDragHandler: Menangani drag and drop pada title bar untuk memindahkan jendela
ResizeHandler: Menangani resize jendela dari arah kanan bawah. menggunakan ikon svg: svgPath.setContent("M542.72 884.053333l341.333333-341.333333a32 32 0 0 1 47.445334 42.816l-2.197334 2.432-341.333333 341.333333a32 32 0 0 1-47.466667-42.837333l2.197334-2.432 341.333333-341.333333-341.333333 341.333333z m-437.333333-10.666666l778.666666-778.666667a32 32 0 0 1 47.445334 42.816l-2.197334 2.432-778.666666 778.666667a32 32 0 0 1-47.466667-42.837334l2.197333-2.432 778.666667-778.666666-778.666667 778.666666z");

4. Package id.alphareso.meidofx.base.sidebars

SideBar: Menyediakan panel navigasi vertikal yang dapat disesuaikan dengan dukungan untuk item navigasi berikon

5. Package id.alphareso.meidofx.base.stages

BaseStage: Kelas dasar untuk semua jendela aplikasi, mewarisi dari Stage JavaFX
RoundStage: Kelas jendela dengan sudut membulat yang mencakup title bar, sidebar opsional, dan area konten

6. Package id.alphareso.meidofx.base.titles

TitleBar: Menyediakan title bar yang dapat disesuaikan dengan ikon, teks judul, dan tombol kendali jendela
